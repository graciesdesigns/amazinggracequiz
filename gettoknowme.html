Why did I create AGQ?

I created the Amazing Grace Quiz in 2022. I was a incoming junior in high school and I was just figuring out I wanted to study computer science in college. Just to test if I would like coding, I did a course on HTML and through that, I had an idea to build my own project. Little by little, I practiced on making HTML forms and using CSS to style and edit my webpage. After 3 years of putting the project in hiatus, the summer before my 2nd year of college, I decided to jump right back into it, to develop and learn more front-end skills and fundementals. I learned many Javascript uses and DOM manipulation along with many CSS tags that can change the way my web page looks. It took a lot of dedication and I am very proud of how far I've come with my very first project. I hope you see the vision like I did.
